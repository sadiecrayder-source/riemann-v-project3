Historical Facts

  • Riemann published his work on integration in 1854.
  • Archimedes used the "Method of Exhaustion" in 250 BC, a precursor to Riemann sums.
  • Leibniz created the "S" symbol (integral) from the word "Summa".
  • Cauchy was the first to define the integral of a continuous function using limits.
  • A function must be bounded to be Riemann integrable.
  • The error in a Trapezoidal sum is proportional to the second derivative.
