Theoretical Architecture

The Limit Definition

Integration is the infinite summation of rectangles. Formally:

\[ \int_{a}^{b} f(x) \, dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i^*) \Delta x \]

On the AP Exam, remember: if the limit exists, the function is Riemann Integrable. This requires the function to be bounded and continuous (or have only finitely many jump discontinuities).

Monotonicity & Estimation

Predicting over/under estimates is a core AP Skill:

  • Increasing Functions: RRAM is an overestimate. LRAM is an underestimate.
  • Decreasing Functions: LRAM is an overestimate. RRAM is an underestimate.

Trapezoidal Accuracy

The Trapezoid Rule uses the average of the Left and Right sums. Its error is directly related to the concavity (second derivative) of the function.

\[ T_n = \frac{\Delta x}{2} [f(x_0) + 2f(x_1) + \dots + f(x_n)] \]
